# appsite-leyu-archive

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常以静态形式呈现，适用于快速浏览、本地保存或作为离线资料使用。仓库本身不针对任何特定域名或网站，仅作为 HTML 页面的集中存放与版本管理空间。

## 目录说明

```
/
├── pages/          # 存放各个独立 HTML 页面文件
├── assets/         # 页面引用的资源文件（图片、样式表、脚本等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件代表一个归档页面。
- `assets/` 目录用于存放页面所需的公共或私有资源，避免路径混乱。

## 页面归档说明

归档的 HTML 页面均保持原始结构与内容，未做功能修改或外部链接植入。每个页面均以独立文件形式存在，可双击打开或部署到任意静态服务器使用。

部分页面可能依赖外部资源（如 CDN 库），若需完全离线使用，建议下载对应资源并放置于 `assets/` 目录下，同时调整页面中的引用路径。

## 维护说明

- 本仓库不定期更新，新增或修正页面内容。
- 欢迎通过 Issue 提交页面归档建议或报告链接失效问题。
- Pull Request 需确保新增页面不包含恶意代码、跟踪脚本或商业推广内容。
- 所有页面仅作存档用途，请遵守相关法律法规及原始内容的版权要求。

## 许可

本仓库内容仅供学习、存档与参考使用。各 HTML 页面的版权归原始作者所有，若需商用或二次发布，请自行联系版权方。仓库整体结构及说明文档采用 MIT 许可协议。
